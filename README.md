# Clinical-Trial-Protocol-Design-Support
Demonstrate how Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG) can:  Extract structured features (disease, drug, eligibility criteria, endpoints) from trial protocols.  Retrieve relevant historical trial information.  Predict trial feasibility metrics.

## Streamlit UI
Run a minimal chat UI over a protocol PDF:

```bash
streamlit run streamlit_app.py
```
